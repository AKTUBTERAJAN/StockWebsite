# StockWebsite
  # Name: TradePath
            TradePath is a full-stack stock trading and investment platform designed to provide users with a secure and user-friendly experience for managing investments. The application includes features such as user authentication, investment dashboard, portfolio management, market insights, and account management.

# Project Structure
    The project is organized into three main folders:

#    📁 Frontend

    The frontend is built using React.js and provides a responsive user interface where users can register, log in, explore investment products, and manage their portfolios.

#    📁 Dashboard

    The dashboard is an admin panel that allows administrators to manage users, monitor transactions, update investment products, and oversee platform activities.

#    📁 Backend

    The backend is developed using Node.js, Express.js, and MongoDB. It handles authentication, APIs, database operations, and secure communication between the frontend and the database.

#  📁 Dashboard
    Provides an overview of the user's portfolio, account balance, investment performance, and recent market activities.

#       Orders
        Allows users to place, view, and manage buy and sell orders with real-time order status.

#       Holdings
        Displays all stocks and investments currently owned by the user, including quantity, average price, and current market value.

#       Positions
        Shows active trading positions with details such as profit/loss, quantity, and market performance.

#       Funds
        Enables users to add, withdraw, and monitor available account funds and transaction history.

# 📁 Backend
        The Backend is built using Node.js, Express.js, and MongoDB to provide a secure and scalable server-side architecture. It is responsible for handling API requests, user authentication using JWT, database management, and business logic.

#       Key Responsibilities
        User Authentication & Authorization (JWT)
        RESTful API Development
        MongoDB Database Integration
        User, Portfolio, and Order Management
        Secure Password Hashing with bcrypt
        CRUD Operations
        Error Handling & Validation
        Protected Routes
        CORS & Middleware Configuration
        Environment Variable Management
